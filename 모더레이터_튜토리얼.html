QR 코드로 참가자 초대

모더레이터로 입장하면 곧바로 QR 코드 모달이 자동으로 열립니다. 이 창을 화면에 띄워 두거나 URL을 복사해서 참가자들에게 공유하세요.

참가자 공유 방법 (셋 중 하나 선택):

  1. QR 코드 스캔 — 스마트폰 카메라로 스캔하면 자동 입장

불참 역할 대리 입력 — 프록시 기능

참가자가 없는 역할의 응답을 모더레이터가 직접 입력할 수 있습니다.

접속하지 않은 역할(예: 보건교사가 없을 때)이 있으면 모더레이터 화면에 보라색 프록시(대리 입력) 패널이 나타납니다.

프록시 (대리 입력) 패널

🔄 대리 입력 (참가자 없는 역할)

접속하지 않은 역할의 응답을 모더레이터가 직접 입력해 피드백에 포함시킬 수 있습니다.
🩺 보건(담당)교사 — 대리 입력
미션: 담임교사와 진한이가 보건실에 도착했습니다. 보건교사로서 어떻게 조치해야 할까요?
① 교사와 진한이가 보건실에 도착한 즉시 손을 씻도록 요청한다.
② 보건교사는 마스크를 착용하고, 체온 측정과 더불어 증상을 관찰한다...
응답을 입력하거나 위 모범답안을 복사해 붙여넣으세요
  1. [📄 모범답안 보기 ▼] 버튼을 클릭하면 해당 역할의 모범 답안이 펼쳐집니다.
  2. [📋 복사] 버튼으로 모범답안을 클립보드에 복사합니다.
  3. 텍스트 입력창에 붙여넣기(Ctrl+V)하거나 직접 입력합니다.
  4. [대리 제출] 버튼을 클릭하면 해당 역할의 응답이 피드백에 포함됩니다.
모범답안을 그대로 복사해서 제출해도 됩니다. 이렇게 하면 피드백 화면에서 전체 역할의 체크리스트를 확인할 수 있습니다.

단계별 피드백 — 체크리스트 채점 및 제출

피드백 화면에서는 역할별로 응답을 확인하고, 모더레이터가 직접 체크할 수 있는 항목이 표시됩니다. 채점 후 [✅ 제출하기]를 누르면 참가자 화면에 점수가 공개됩니다.

참가자 화면 — 역할 필터 기능

참가자는 피드백 화면 상단의 필터 버튼으로 보고 싶은 역할만 선택해서 볼 수 있습니다.

전체 나 담임 보건 행정실 교장/교감 교육청

피드백 진행 순서:

  1. 각 역할 카드를 순서대로 열어 응답 내용을 큰 화면에 띄워 함께 확인
  2. 키워드를 클릭해 체크, 자동 감지 항목도 확인 후 수정 가능
  3. 채점이 끝나면 [✅ 제출하기] 클릭 → 참가자에게 점수 공개
  4. 빠진 항목은 왜 중요한지 설명하며 모범 답안으로 보충 설명
  5. 토론이 끝나면 컨트롤 패널에서 다음 단계 시작
8

단계 전환 — 피드백 확인 후 다음 단계로

단계가 끝나면 피드백을 보고 역할을 재배정한 뒤 다음 단계를 시작합니다.

타이머가 종료되거나 [⏭ 지금 피드백]을 누르면 단계별 피드백 화면이 열립니다. 피드백 논의를 마친 후 아래 버튼을 눌러 다음 단계로 넘어갑니다.

1단계 피드백 후 — 모더레이터 컨트롤

🎙️ 진행자 컨트롤 — 1단계 피드백 중

📋 다음 단계 참여 역할: 📚담임교사, 🩺보건(담당)교사, 🏛️교장/교감, 🏢교육(지원)청
(2단계: 🚨 확진자 발생 및 능동감시)

  1. 피드백 화면에서 각 역할의 응답과 키워드 체크리스트를 함께 논의합니다.
  2. 다음 단계에서 새 역할이 필요하면 역할 재배정 드롭다운을 조정합니다. (예: 교장/교감 역할을 새 참가자에게 배정)
  3. [📢 역할 재배정 공지하기]를 누르면 참가자 화면에 새 역할이 공지됩니다.
  4. 분 수를 확인하고 [▶ 2단계 시작]을 클릭합니다. — 클릭 즉시 타이머가 자동 시작됩니다.
역할 재배정 없이 동일 역할로 계속 진행하려면 드롭다운을 수정하지 않고 [▶ 2단계 시작]만 눌러도 됩니다.
[◀ 피드백 종료]를 누르면 피드백 화면을 닫고 해당 단계 진행 화면으로 돌아갑니다. 1단계 진행 화면에서 다시 [◀]를 누르면 대기실로 돌아갈 수 있습니다.
2단계(확진자 발생)에는 교장/교감과 교육(지원)청이 추가됩니다. 해당 역할 참가자가 없으면 프록시 패널에서 대리 입력해 주세요.
